# FlowGym — Guia rápido de marca

## Nome

FlowGym

## Paleta

- Teal fluxo: `#0EA5A0`
  - Marca, logo, header, botões principais.
- Azul profundo: `#0B2545`
  - Sidebar, áreas escuras e elementos de robustez.
- Verde elétrico: `#C6F135`
  - CTAs pontuais, alertas positivos, metas batidas e destaques de progresso.
- Azul céu: `#38BDF8`
  - Gráficos, barras de progresso e ícones secundários.
- Grafite: `#111827`
  - Texto principal.
- Cinza gelo: `#F8FAFC`
  - Fundo base do app.

## Uso no app

- Sidebar usa azul profundo.
- Botões principais usam teal fluxo.
- Gráficos e barras usam teal + azul céu.
- Destaques positivos usam verde elétrico com moderação.
- Cards e tabelas permanecem claros para leitura.
